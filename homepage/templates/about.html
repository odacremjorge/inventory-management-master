Almacen - Oruro
Un sistema de gestión de inventario


| Acerca de Nosotros |


Almacen de Oruro aplica este software de gestión de inventario simple, creado para los requisitos de la empresa ENDE Servicios y Construcciones S.A. en su Regional Oruro. El software es capaz de realizar un seguimiento de las existencias de repuestos entrantes y salientes con un comprobante renderizado dinámicamente. A cada proveedor se le proporciona un perfil donde puede ver el historial de sus transacciones con la empresa.


| Caracteristicas |


  • Interfaz interactivo para mostrar el stock disponible actual
  • Se mantienen los detalles de stock, como el stock disponible actual
  • Los repuestos entrantes se realizan a través de compras o por envio de la central en La Paz.
  • Los repuestos salientes se realizan a través de salidas.
  • Cada transacción realizada a través del software se registra
  • El comprobante de salida se renderiza dinámicamente y se puede imprimir
  • Se mantienen perfiles separados para los proveedores
